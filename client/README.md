# Client Frontend - Crypto Savings Tracker

Folder ini akan berisi aplikasi frontend React untuk Crypto Savings Tracker.

## 📌 Status

**Saat ini**: Belum dimulai  
**Target**: Phase 2-3 development

## 📋 Rencana

### Phase 2: React Setup

- Initialize project dengan Create React App atau Vite
- Setup folder structure
- Setup routing
- Setup styling

### Phase 3: UI Components

- Membuat component library
- Konversi dari HTML prototype ke React components
- Setup state management

### Phase 4: Logic & State

- Pindahkan transaction logic dari vanilla JS
- Setup Context API / Redux
- Integrate dengan backend API

## 🛠️ Tech Stack (Rencana)

- **Framework**: React 18+
- **Router**: React Router v6
- **Styling**: Tailwind CSS / CSS Modules
- **State**: Context API atau Redux
- **HTTP Client**: Axios atau Fetch API
- **Testing**: Jest + React Testing Library
- **Build**: Vite
