Folder ini akan berisi API backend Node.js/Express untuk Crypto Savings Tracker.

**Saat ini**: Belum dimulai  
**Target**: Phase 5-6 development

## 📋 Rencana

### Phase 5: Express Setup

- Initialize Node.js project
- Setup Express server
- Setup routing
- Setup middleware

### Phase 6: API Development

- Buat endpoints untuk transactions
- Buat endpoints untuk assets
- Setup database connection
- Implement authentication

### Phase 7: Database

- Design schema
- Setup PostgreSQL atau MongoDB

## 🛠️ Tech Stack (Rencana)

- **Runtime**: Node.js 18+
- **Framework**: Express.js
- **Database**: PostgreSQL atau MongoDB
- **ORM**: Sequelize, TypeORM, atau Prisma
- **Authentication**: JWT
- **Validation**: Joi atau Yup
- **Testing**: Jest + Supertest
- **Environment**: dotenv
